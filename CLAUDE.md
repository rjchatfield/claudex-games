# Pokemon Games

This contains the tracking information for play throughs of a specific Pokemon game. Together with the Claude Code Plugin, Claude Code is able to be interacted with like a Rotom-powered Pokedex. Giving advice on which pokemon to catch, which moves to learn, how to beat opponents, and more.

Each directory represents a specific playthrough of a game, and should follow the same exact folder structure.

When begining a new playthrough:
- Create a unique folder
- Setup the Claude Code plugin from `../plugin`
- Run the `/claudex:init` command in Claude Code
